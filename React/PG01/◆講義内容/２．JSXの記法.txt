◆JSXとは、JavaScriptの拡張言語

JSXは
---------------------------------------------
return (
    <button className={'btn-blue'}>
    Click me!
)
---------------------------------------------
内の記述のことを言う



上記をJSXを使わないと、
---------------------------------------------
React.createElement(
    'button',
    {className: 'btn-blur'}.
    'Click me!
)
---------------------------------------------
になる。




JSXの基礎文法①
１．Reactライブラリをimportする
２．retun文の中にJSXを記載する
　　※class名を記述するときは「className」とする必要がある

JSXの基礎文法②
１．キャメルケースで記載する
２．{}で囲むことで変数を使用することができる

JSXの特殊な構文
１．JSXは階層構造でなければならない

NG
---------------------------------------------
return (
    <p>Hello</p>
    <p>World</p>
)
---------------------------------------------


OK
---------------------------------------------
return (
    <React.Fragment>
        <p>Hello</p>
        <p>World</p>

---------------------------------------------


OK
---------------------------------------------
return (
    <>
        <p>Hello</p>
        <p>World</p>
    </>
)
---------------------------------------------
